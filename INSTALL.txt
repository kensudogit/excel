========================================
Excel キーワード検索アプリケーション
インストール手順
========================================

【必要な環境】
- Python 3.8以上
- Node.js 16以上

【セットアップ手順】

1. このフォルダを任意の場所にコピーしてください

2. セットアップスクリプトを実行してください
   Windows: setup.bat をダブルクリック
   Linux/Mac: ./setup.sh を実行

3. 起動スクリプトを実行してください
   Windows: run.bat をダブルクリック
   Linux/Mac: ./run.sh を実行

4. ブラウザで http://localhost:3001 にアクセスしてください

【注意事項】
- 初回セットアップには数分かかる場合があります
- インターネット接続が必要です（依存関係のダウンロード）
- ファイアウォールの警告が表示された場合は許可してください
